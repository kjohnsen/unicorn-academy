# Technical Requirements and Guidelines

- **Framework**: Use Svelte 5.
- **Language**: Use TypeScript for all logic and components.
- **Responsiveness**: The application must work seamlessly on both desktop and mobile browsers. Ensure all UI elements are touch-friendly and layouts adapt to different screen sizes.
- **Runtime & Package Manager**: Use Bun for everything (e.g., `bun install`, `bun run`, etc.).
