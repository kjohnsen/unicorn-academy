# Unicorn Academy - Design Specification

## Overview
A web-based game where players interact with girls and their unicorns at the Unicorn Academy. The app features a stable environment where users can view and customize both the girls and their unicorns.

## Core Screens & Interactions

### 1. Main View (The Stable)
- Displays a visual representation of a stable.
- Shows multiple girls and multiple unicorns within the stable environment.
- Two main interaction buttons are present on the screen:
  - **"View Unicorns"**: Opens the Unicorn List view.
  - **"View Girls"**: Opens the Girls List view.

### 2. Unicorn Management
- **Unicorn List**: A view triggered from the Main View. Displays a list or grid of all unicorns, along with their current names.
- **Unicorn Customization**: Clicking on a specific unicorn in the list opens a customization interface.
  - Users can edit the unicorn's **name**.
  - Users can alter the unicorn's **design** (e.g., colors, markings, accessories).

### 3. Girl Management
- **Girls List**: A view triggered from the Main View. Displays a list or grid (tiles) of all the girls, along with their current names.
- **Girl Customization**: Clicking on a specific girl's tile opens a customization interface.
  - Users can edit the girl's **name**.
  - Users can customize the girl's **hair** (style and color).
  - Users can customize the girl's **clothes** (outfits and accessories).
